# Java Script functions 
    * alert(document here);    // for alert when page is loaging
    * document.write("Helo Java Script <br/>");    // To write in document page
    * console.log(document  here);    // To write in console. 
    * typeof(variable);    // To show variable type. 
    * toString(num);    // for convert number to string. 
    * parseInt(name);    // for convert string to number. 
    * Number("124.55");    // for convert to number. 
    * num.toFixed(2);    // for output two digits after decimal point. 
    * num.toPrecision(3);    // for output total 3 digit. 

    * name.length;     // for string length. notice there is no paranthesis. 
    * name = prompt("Enter your name: ");    // for input from user with massage. 
    * name.charAt(2);    // for access third digit of name.
    * toUpperCase(name);    // for convert name to uppercase. 
    * name.concat(lastName);    // for concatenate name and lastName string. 
    * sub = name.slice(2, 5);    // for substring from 3rd index up to 6th index.
